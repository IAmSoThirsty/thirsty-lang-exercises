Practice Thirsty-lang with hands-on coding exercises. From beginner to advanced!

## 📚 Exercise Categories

### 🟢 Beginner (10 exercises)

Perfect for newcomers to Thirsty-lang.

### 🟡 Intermediate (10 exercises)

Build on fundamentals with more complex problems.

### 🔴 Advanced (10 exercises)

Challenge yourself with real-world scenarios.

## 🚀 How to Use

1. **Choose your level**
2. **Read the exercise description**  
3. **Write your solution**
4. **Check the solution** (in `solutions/` folder)
5. **Move to next exercise**

## 💡 Tips for Success

1. **Try before looking** - Attempt the solution yourself first
2. **Read the error messages** - They guide you to the problem
3. **Experiment** - Try different approaches
4. **Study solutions** - Learn from working code
5. **Practice daily** - Consistency builds skills
